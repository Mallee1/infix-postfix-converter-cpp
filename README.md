# Infix to Postfix Converter (C++)

## Overview
This project converts infix expressions to postfix notation and evaluates postfix expressions using a custom generic stack implementation in C++.

## Features
- Generic stack implemented with C++ templates
- Infix to postfix conversion
- Postfix expression evaluation
- Demonstrates parsing and algorithmic problem solving

## Technologies Used
- C++
- Object-oriented programming
- Data structures and algorithms

## Files
- `GenericStack.h` - stack class definition
- `GenericStack.cpp` - stack implementation
- `in2post.cpp` - main program for conversion and evaluation

## What I Learned
Through this project, I strengthened my understanding of:
- stack data structures
- template programming in C++
- expression parsing
- postfix evaluation logic

## How to Run
1. Open the project in your C++ IDE or compiler
2. Compile the source files
3. Run the program
4. Enter or test infix expressions for conversion and evaluation
